#### **Част 2: Основни компоненти (20 мин)**

**1. ЦЕНТРАЛЕН ПРОЦЕСОР (CPU):**

**Структура:**
- **ALU (Arithmetic Logic Unit)** - аритметични и логически операции
- **CU (Control Unit)** - управление и координация
- **Регистри** - временно съхранение
- **Кеш памет** - бърз достъп до данни

**Демонстрация с CPU-Z:**
```bash
# Информация за процесора:
- Модел и производител
- Честота (base и boost)
- Брой ядра и нишки
- Кеш памет (L1, L2, L3)
- Инструкционни набори
```

**2. ОПЕРАТИВНА ПАМЕТ (RAM):**

**Характеристики:**
- Волатилна (временна)
- Произволен достъп
- Високоскоростна
- Работно пространство

**Типове:**
- SRAM - Static RAM (кеш)
- DRAM - Dynamic RAM (основна)
- DDR4/DDR5 - съвременни стандарти

**3. ВХОДНО-ИЗХОДНИ УСТРОЙСТВА:**

**Входни:**
- Клавиатура, мишка
- Скенер, камера
- Микрофон, сензори

**Изходни:**
- Монитор, принтер
- Колони, LED индикатори

**Двупосочни:**
- Мрежови карти
- USB портове
- Touchscreen дисплеи

#### **Част 3: Системни шини и комуникация (10 мин)**

**Дейност на учителя:**
- Обяснение на концепцията за шини
- Демонстрация с реална дънна платка

**Типове шини:**
1. **Адресна шина** - указва местоположение
   - Широчина определя максималната памет
   - 32-bit = 4GB, 64-bit = 16EB

2. **Шина за данни** - пренася информация
   - Широчина определя скоростта
   - 64-bit в съвременните системи

3. **Управляваща шина** - контролни сигнали
   - Read/Write сигнали
   - Interrupt signals
   - Clock signals

**Информационен поток - пример:**
```
1. CPU изпраща адрес по адресната шина
2. Изпраща Read сигнал по управляващата шина
3. RAM намира данните на този адрес
4. RAM изпраща данните по шината за данни
5. CPU получава и обработва данните
```

**Дейност на учениците:**
- Проследяване на примера
- Рисуване на диаграма на потока
- Идентифициране на шините на дънната платка

### **4. ЗАТВЪРЖДАВАНЕ (20 минути)**

#### **Практическа работа: Анализ на системата**

**Задача 1: Използване на диагностичен софтуер**
```bash
# Стъпки:
1. Стартирайте CPU-Z
2. Запишете информацията:
   - Процесор: модел, честота, ядра
   - Памет: тип, размер, честота
   - Дънна платка: модел, чипсет

3. Стартирайте Task Manager
4. Наблюдавайте:
   - CPU натоварване
   - RAM използване
   - Процеси и услуги
```

**Задача 2: Изчисления**
1. Ако адресната шина е 20-битова, каква е максималната адресируема памет?
   - Отговор: 2²⁰ = 1,048,576 байта = 1MB

2. Колко данни може да прехвърли 64-битова шина за един такт?
   - Отговор: 64 бита = 8 байта
