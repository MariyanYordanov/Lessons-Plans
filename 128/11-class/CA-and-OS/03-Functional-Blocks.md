### **3. НОВ МАТЕРИАЛ (40 минути)**

#### **Част 1: Процесорен блок (10 мин)**

**Основни параметри:**
```
1. Честота (Clock Speed)
   - Base frequency: 3.6 GHz
   - Boost frequency: 5.0 GHz
   - Формула: Operations/sec = Frequency × IPC

2. Ядра и нишки
   - Physical cores: 8
   - Logical threads: 16 (с Hyper-Threading)
   - Паралелизъм и многозадачност

3. Кеш памет
   - L1: 32KB per core (най-бърза)
   - L2: 256KB per core
   - L3: 16MB shared (най-бавна от кеша)

4. Технологичен процес
   - 7nm, 10nm, 14nm
   - По-малък = по-ефективен

5. TDP (Thermal Design Power)
   - 65W, 95W, 125W
   - Определя охлаждането
```

**Примерни изчисления:**
- CPU с 4 ядра @ 3.5 GHz, IPC = 4
- Теоретична производителност = 4 × 3.5 × 4 = 56 GIPS

#### **Част 2: Памет и йерархия (10 мин)**

**Йерархия на паметта:**
```
Регистри     < 1ns     < 1KB       Най-бърза
L1 Cache     ~ 1ns     32-64KB     
L2 Cache     ~ 3ns     256KB-1MB   
L3 Cache     ~ 10ns    8-32MB      
RAM          ~ 100ns   8-64GB      
SSD          ~ 0.1ms   256GB-2TB   
HDD          ~ 10ms    1-10TB      Най-бавна
```

**RAM параметри:**
```
DDR4-3200 спецификация:
- Честота: 3200 MHz
- Bandwidth: 25.6 GB/s (3200 × 8 bytes)
- Латенция: CL16-18-18-38
- Voltage: 1.35V

Dual Channel изчисление:
- Single: 25.6 GB/s
- Dual: 51.2 GB/s
```

## 8. ПРИЛОЖЕНИЯ

### ПРИЛОЖЕНИЕ 1: ФОРМУЛИ И ИЗЧИСЛЕНИЯ

```
Memory Bandwidth = Frequency × Bus Width × Channels
                 = 3200 MHz × 64 bit × 2 / 8
                 = 51.2 GB/s

Storage IOPS = Queue Depth × 1000 / Latency(ms)

Power Consumption = TDP × Load % × Inefficiency
                  = 125W × 0.8 × 1.15
                  = 115W real

Temperature Rise = Power / Cooler Efficiency
                 = 125W / 2.5 W/°C
                 = 50°C above ambient
```
