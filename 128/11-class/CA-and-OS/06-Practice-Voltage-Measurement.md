## 5. ХОД НА УРОКА

### **1. ОРГАНИЗАЦИОНЕН МОМЕНТ (5 минути)**

**Дейност на учителя:**
- Проверка на присъствието
- Разпределение по работни места
- **КРИТИЧНО:** Инструктаж за безопасност

**Правила за безопасност:**
```
⚠️ ЗАДЪЛЖИТЕЛНИ ПРАВИЛА:
1. НИКОГА не измервайте напрежение в режим на съпротивление!
2. Първо свързвайте черния (COM) проводник
3. Започвайте с най-високия обхват
4. Не докосвайте голи проводници
5. Изключвайте захранването преди промени
6. Внимавайте с кондензаторите - запазват заряд!
```

**Дейност на учениците:**
- Организиране на работните места
- Проверка на инструментите
- Подписване на инструктажа

### **2. АКТУАЛИЗАЦИЯ НА ЗНАНИЯТА (10 минути)**

**Дейност на учителя:**
- Демонстрация на основни измервания

**Практически преговор:**
1. Как се настройва мултимер за DC напрежение?
2. Кои са стандартните напрежения в ATX?

### **3. ПРАКТИЧЕСКА РАБОТА (65 минути)**

#### **Станция 1: Основни измервания с мултимер (20 мин)**

**Задача 1.1: Тестване на батерии и адаптери**

**Процедура:**
```
Обекти за измерване:
- 9V батерия
- AA/AAA батерии (1.5V)
- Литиева батерия (3V)
- USB зарядно (5V)
- Лаптоп адаптер (19V)

Стъпки:
1. Изберете подходящ обхват
2. Свържете правилно сондите
3. Измерете и запишете
4. Определете състоянието
```

**Протокол за измерване:**
| Обект | Номинално | Измерено | Състояние |
|-------|-----------|----------|-----------|
| 9V батерия | 9.0V | | |
| AA нова | 1.5V | | |
| AA използвана | 1.5V | | |
| CR2032 | 3.0V | | |
| USB charger | 5.0V | | |

**Критерии за оценка:**
- Нова батерия: >90% от номинала
- Използваема: 75-90%
- За смяна: <75%

**Задача 1.2: Измерване на ATX конектор**

**24-pin ATX pinout за измерване:**
```
Pin | Цвят    | Напрежение | Измерено | OK?
----|---------|------------|----------|-----
1   | Оранжев | +3.3V      |          |

Толеранси:
```

#### **Станция 2: Тестване под натоварване (20 мин)**

**Задача 2.1: Статично натоварване**

**Изкуствен товар:**
```
Създаване на товар:
- За +5V: 10Ω 10W резистор = 0.5A
- За +12V: 27Ω 10W резистор = 0.44A

Свързване:
1. Molex конектор
2. Резистор между +12V (жълт) и GND (черен)
3. Паралелно свързан мултимер

Измервания:
- Напрежение без товар: _____V
- Напрежение с товар: _____V
- Спад на напрежение: _____V
- Приемливо ако <5% спад
```

**Задача 2.2: Динамично натоварване**

**Тест с реални компоненти:**
```
Setup:
1. Свържете HDD/SSD
2. Измерете +5V и +12V на SATA power
3. Стартирайте benchmark
4. Наблюдавайте промените

Протокол:
| Състояние | +5V | +12V | Забележки |
|-----------|-----|------|-----------|
| Idle | | | |
| Startup spike | | | |
| Load | | | |
| Shutdown | | | |
```

#### **Станция 3: Осцилоскоп измервания (15 мин)**

**Задача 3.1: Ripple измерване**

**Настройка на осцилоскопа:**
```
Настройки:
- Coupling: AC (за ripple)
- Voltage: 20mV/div
- Time: 10μs/div
- Trigger: Auto

Процедура:
1. Свържете към +12V rail
2. GND към черен проводник
3. Наблюдавайте AC component
4. Измерете Vpp (peak-to-peak)

Допустими стойности:
+12V: <120mVpp
+5V:  <50mVpp
+3.3V: <50mVpp
```

**Задача 3.2: PWM сигнал на вентилатор**

```
4-pin вентилатор:
Pin 1: GND (черен)
Pin 2: +12V (червен)
Pin 3: Tach (жълт)
Pin 4: PWM (син)

Измерване на PWM:
- Frequency: ~25kHz
- Duty cycle: 20-100%
- Voltage: 5V TTL
```

#### **Станция 4: Диагностика на проблеми (10 мин)**

**Задача 4.1: Откриване на дефекти**

**Сценарии за диагностика:**

**Сценарий A: "Компютърът не стартира"**
```
Проверки:
1. AC входно напрежение (220V)
2. +5VSB standby (лилав)
3. PS_ON сигнал (зелен) - 0V при старт
4. PWR_OK сигнал (сив) - 5V ако OK

Възможни проблеми:
- Липсва +5VSB → дефектен standby
- PS_ON не пада → проблем с MB/бутон
- Липсва PWR_OK → PSU защита
```

**Сценарий B: "Рандом рестарти"**
```
Проверки:
1. Стабилност на +12V под товар
2. Ripple на всички rails
3. Температура на PSU
4. Капацитет достатъчен ли е

Измервания:
- Min/Max напрежения
- Transient response
- Voltage droop
```

**Сценарий C: "Артефакти на екрана"**
```
GPU power проверки:
1. PCIe 6/8-pin конектори
2. Всички +12V pins свързани
3. Напрежение >11.4V под товар
4. Достатъчен ток на +12V rail
```

### ПРИЛОЖЕНИЕ 3: ДИАГНОСТИЧНИ ПРОЦЕДУРИ

**Flowchart за диагностика:**
```
Компютър не стартира
        ↓
[Проверка AC вход]
   220V? → НЕ → Проверка кабел/контакт
     ↓ДА
[Проверка +5VSB]
   5V? → НЕ → PSU дефект (standby)
     ↓ДА
[Проверка PS_ON]
   0V при натискане? → НЕ → Проблем MB/бутон
     ↓ДА
[Проверка основни rails]
   В толеранс? → НЕ → PSU дефект/претоварване
     ↓ДА
[Проверка PWR_OK]
   5V? → НЕ → PSU в защита/нестабилен
     ↓ДА
[Проблемът е в друг компонент]
```

**Quick Test процедура:**
```bash
# Paper clip test
1. Изключете PSU от мрежата
2. Свържете зелен (PS_ON) с черен (GND)
3. Включете в мрежата
4. PSU трябва да стартира
5. Измерете всички напрежения

# Load test
1. Свържете известен добър товар
2. Измерете напреженията
3. Наблюдавайте за 10 минути
4. Проверете температурата
5. Ripple тест с осцилоскоп
```
