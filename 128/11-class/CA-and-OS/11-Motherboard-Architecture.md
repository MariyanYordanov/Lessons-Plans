### **3. НОВ МАТЕРИАЛ (40 минути)**

#### **Част 1: Какво е дънна платка? (8 мин)**

**Определение:**
```
Дънната платка (motherboard) е основната печатна платка 
в компютъра, която:
- Осигурява физическа и електрическа връзка между компонентите
- Координира комуникацията чрез чипсета
- Разпределя захранването
- Предоставя интерфейси за периферни устройства
```

**Аналогия:**
"Дънната платка е като централния площад на град - всички пътища водят до нея и през нея преминава цялата комуникация."

#### **Част 2: Форм-фактори (7 мин)**

| Форм-фактор | Размери (mm) | PCIe слотове | RAM слотове | Приложение |
|-------------|--------------|--------------|-------------|------------|
| E-ATX | 305×330 | 7 | 8 | Сървъри, Workstations |
| ATX | 305×244 | 4-6 | 4 | Стандартни настолни PC |
| Micro-ATX | 244×244 | 2-4 | 2-4 | Компактни системи |
| Mini-ITX | 170×170 | 1 | 2 | HTPC, малки системи |

**Демонстрация:**
- Показване на реални платки от различни форм-фактори
- Сравняване на размери и възможности

#### **Част 3: Основни компоненти (15 мин)**

**1. Процесорен сокет:**
```
Intel:
├── LGA 1700 - 12-14 gen Core (Alder Lake, Raptor Lake)
├── LGA 1851 - Core Ultra (Arrow Lake) - 2024+
└── LGA 4677 - Xeon сървърни

AMD:
├── AM5 - Ryzen 7000/9000 серия (DDR5)
├── AM4 - Ryzen 1000-5000 (DDR4)
└── TR5 - Threadripper PRO
```

**2. Чипсет - "Мозъкът" на платката:**
```
Intel Z790 Chipset:
├── До 20 PCIe 4.0 lanes
├── До 8 SATA 6Gbps порта
├── До 5 USB 3.2 Gen 2x2 (20Gbps)
├── Intel Rapid Storage Technology
└── WiFi 6E поддръжка

AMD X670E Chipset:
├── До 44 PCIe 5.0/4.0 lanes
├── До 8 SATA 6Gbps
├── До 14 USB ports
├── Dual chiplet design
└── PCIe 5.0 за GPU и NVMe
```

**Block диаграма (на дъска/екран):**
```

   └─────────┘ └───────┘ └───┘ └───────┘ └─────────┘
```

**3. Слотове за памет (DIMM):**
```
DDR5 DIMM слот характеристики:
- 288 пина (различна конфигурация от DDR4)
- Ключ на различна позиция от DDR4
- Поддръжка на ECC на канала
- On-die ECC на модула
- Power Management IC (PMIC) на модула
```

**4. PCIe слотове:**
```
PCIe 5.0 x16: 64 GB/s - за видеокарти
PCIe 4.0 x4:  8 GB/s  - за NVMe SSD
PCIe 3.0 x1:  1 GB/s  - за разширителни карти
```

**5. VRM (Voltage Regulator Module):**
```
Функция: Преобразува 12V от захранването в напрежение за CPU

Компоненти:
├── MOSFETs - превключващи транзистори
├── Chokes (дросели) - филтрират тока
├── Capacitors - изглаждат напрежението
└── PWM Controller - управлява фазите

Брой фази:
- Budget: 4-6 фази
- Mainstream: 8-12 фази
- High-end: 14-24+ фази
```

#### **Част 4: Конектори и портове (10 мин)**

**Вътрешни конектори:**
```
Захранване:
├── 24-pin ATX main - основно захранване
├── 8-pin EPS12V - CPU захранване
└── 6-pin PCIe (понякога) - допълнително

Съхранение:
├── SATA - 7+15 пина (данни + захранване)
├── M.2 - 2242, 2260, 2280, 22110 размери
└── U.2 - enterprise NVMe

Допълнителни:
├── Front panel (power, reset, LED)
├── USB 2.0/3.0/3.2 headers
├── Audio header (HD Audio)
├── Fan headers (CPU, System, Pump)
└── RGB headers (3-pin ARGB, 4-pin RGB)
```

**I/O панел (задни портове):**
```
Типични портове на модерна Z790 платка:
├── USB 3.2 Gen 2x2 Type-C (20 Gbps)
├── USB 3.2 Gen 2 Type-A (10 Gbps) x4
├── USB 2.0 Type-A x2
├── HDMI 2.1 / DisplayPort 1.4
├── 2.5G Ethernet RJ-45
├── WiFi 6E антенни конектори
├── Audio jacks (7.1 канала)
└── BIOS Flash button
```
