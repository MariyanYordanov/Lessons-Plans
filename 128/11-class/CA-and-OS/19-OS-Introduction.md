"Представете си компютър без операционна система. Как бихте стартирали програма?"

**Дефиниция на ОС:**
```
Операционната система е системен софтуер, който:
1. Управлява хардуерните ресурси
2. Предоставя услуги на приложенията
3. Осигурява интерфейс за потребителя
4. Координира изпълнението на програми
5. Осигурява сигурност и защита
```

**Аналогия:**
"ОС е като управител на хотел - разпределя стаи (памет), координира услуги (процеси), осигурява сигурност"

### **3. НОВ МАТЕРИАЛ (50 минути)**

#### **Част 1: История и еволюция на ОС (15 мин)**

**Времева линия:**

```
1940s - Без ОС
├── Ръчно програмиране
├── Превключватели и перфокарти
└── Една програма наведнъж

1950s - Batch системи
├── GM-NAA I/O (1956)
├── Автоматична обработка на задачи
└── Job scheduling

1960s - Multiprogramming
├── CTSS (1961) - time-sharing
├── OS/360 (1964) - IBM
├── UNIX (1969) - Bell Labs
└── Multics проект

1970s - Микрокомпютърна ера
├── CP/M (1974)
├── Apple DOS (1978)
└── Развитие на UNIX

1980s - Персонални компютри
├── MS-DOS (1981)
├── Mac OS (1984) - GUI
├── Windows 1.0 (1985)
└── OS/2 (1987)

1990s - Модерни ОС
├── Linux (1991)
├── Windows 95 (1995)
├── Mac OS X beta (1999)
└── Palm OS, Windows CE

2000s - Мобилна революция
├── Windows XP (2001)
├── Mac OS X (2001)
├── Android (2008)
├── iOS (2007)
└── Cloud OS концепции

2010s - Облачни и IoT
├── Windows 10 (2015)
├── Container OS
├── Chrome OS
└── IoT операционни системи

2020s - AI и Edge Computing
├── Windows 11 (2021)
├── macOS Ventura/Sonoma
├── Android 14/15
└── Специализирани AI OS
```

**2. Управление на памет:**

**4. I/O управление:**
