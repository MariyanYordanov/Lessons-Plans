### **3. ТЕРМИНАЛ И SHELL (10 минути)**

**Терминал vs Shell:**
```
Terminal (Терминал):
- Програмата, която показва текстовия интерфейс
- Пример: GNOME Terminal, Konsole, Windows Terminal

Shell (Обвивка):
- Интерпретаторът на команди
- Bash (Bourne Again Shell) - най-популярен
- Други: zsh, fish, sh
```

**Структура на команда:**
```
command [options] [arguments]

Примери:
ls                    # само команда
ls -l                 # команда + опция
ls -l /home           # команда + опция + аргумент
ls -la /home          # комбинирани опции

Опции:
- Кратка форма: -l, -a, -h
- Дълга форма: --long, --all, --human-readable
- Могат да се комбинират: -la = -l -a
```

**Prompt:**
```
user@hostname:~$

user     - потребителско име
hostname - име на компютъра
~        - текуща директория (~ = home)
$        - обикновен потребител
#        - root потребител (администратор)
```

### **4. ОСНОВНИ КОМАНДИ (35 минути)**

#### **Част 1: Навигация (10 мин)**

```bash
# Къде съм?
pwd                   # Print Working Directory
# /home/user

# Покажи съдържанието
ls                    # List (просто)
ls -l                 # Long format (детайлно)
ls -a                 # All (включително скрити файлове)
ls -la                # Комбинация
ls -lh                # Human readable размери

# Разбиране на ls -l изхода:
# drwxr-xr-x 2 user group 4096 Jan 15 10:30 Documents
# │├──┬───┤ │  │     │     │      │          │
# ││  │   │ │  │     │     │      │          └─ Име
# ││  │   │ │  │     │     │      └─ Дата/час
# ││  │   │ │  │     │     └─ Размер
# ││  │   │ │  │     └─ Група
# ││  │   │ │  └─ Собственик
# ││  │   │ └─ Брой връзки
# ││  └───┴─ Права (owner/group/others)
# │└─ Тип (d=dir, -=file, l=link)

# Придвижване
cd /home              # Change Directory - абсолютен път
cd Documents          # Относителен път
cd ..                 # Нагоре едно ниво
cd ../..              # Нагоре две нива
cd ~                  # Home директория
cd                    # Също home (без аргумент)
cd -                  # Предишна директория

# Специални директории:
# .  = текуща директория
# .. = родителска директория
# ~  = home директория
# /  = root (корен) на файловата система
```

**Упражнение 1:**
```bash
# Изпълнете последователно:
pwd
cd /
ls
cd /etc
ls -l
pwd
cd ~
pwd
```

#### **Част 2: Работа с файлове (12 мин)**

```bash
# Създаване на директория
mkdir mydir                    # Make Directory
mkdir -p dir1/dir2/dir3        # Създай и родителите

# Създаване на празен файл
touch file.txt                 # Създава или обновява timestamp

# Копиране
cp source.txt dest.txt         # Copy
cp file.txt /tmp/              # Копирай в друга директория
cp -r dir1 dir2                # Копирай директория (рекурсивно)

# Преместване/Преименуване
mv old.txt new.txt             # Move/Rename
mv file.txt /tmp/              # Премести в друга директория
mv dir1 dir2                   # Преименувай директория

# Изтриване (ВНИМАНИЕ!)
rm file.txt                    # Remove file
rm -r directory                # Remove directory (рекурсивно)
rm -rf directory               # Force remove (без въпроси)
rmdir emptydir                 # Remove empty directory only

# ⚠️ ВНИМАНИЕ: rm няма кошче! Изтритото е загубено!
```

**Упражнение 2:**
```bash
cd ~
mkdir practice
cd practice
touch file1.txt file2.txt
ls
mkdir subdir
cp file1.txt subdir/
ls subdir
mv file2.txt renamed.txt
ls
rm file1.txt
ls
```

#### **Част 3: Преглед на съдържание (8 мин)**

```bash
# Целия файл
cat file.txt                   # Concatenate and display
cat file1.txt file2.txt        # Покажи няколко файла

# Първите/последните редове
head file.txt                  # Първите 10 реда
head -n 5 file.txt             # Първите 5 реда
tail file.txt                  # Последните 10 реда
tail -n 20 file.txt            # Последните 20 реда
tail -f /var/log/syslog        # Follow - следи за нови редове

# Постраничен преглед
less file.txt                  # По-добър от more
# Навигация в less:
# Space/PgDn = следваща страница
# b/PgUp = предишна страница
# g = начало
# G = край
# /text = търсене
# q = изход
```

#### **Част 4: Помощ и информация (5 мин)**

```bash
# Manual pages
man ls                         # Пълна документация
man cp
# Навигация: Space, q за изход

# Кратка помощ
ls --help                      # Опции и синтаксис
cp --help

# Какво е тази команда?
which python                   # Път до изпълнимия файл
type ls                        # Тип на командата

# История на командите
history                        # Показва история
!50                            # Изпълни команда #50
!!                             # Повтори последната команда

# Autocomplete
# Tab = автоматично допълване
# Tab Tab = покажи възможности
```

cd ../projects/project1

cd ~/linux_practice
