## 5. ХОД НА УРОКА

### **3. НОВ МАТЕРИАЛ (40 минути)**

#### **Сравнителна таблица:**

| Аспект | Linux | Windows |
|--------|-------|---------|
| **Лиценз** | Безплатен, Open Source (GPL) | Комерсиален, платен |
| **Kernel** | Monolithic, Linux kernel | Hybrid, NT kernel |
| **Файлова система** | ext4, XFS, Btrfs | NTFS, FAT32, ReFS |
| **Root директория** | / (единствена) | C:\, D:\... (per drive) |
| **Case sensitivity** | Да (File ≠ file) | Не (File = file) |
| **Изпълними файлове** | Няма разширение (права) | .exe, .msi, .bat |
| **Package manager** | apt, dnf, pacman | MS Store, winget, chocolatey |
| **Shell** | Bash, Zsh | CMD, PowerShell |
| **GUI** | Optional (KDE, GNOME) | Интегриран (Explorer) |
| **Drivers** | Вградени в kernel | Отделни инсталации |
| **Market share** | ~2-3% desktop, 90%+ сървъри | ~75% desktop |

#### **Еквивалентни команди:**

| Задача | Linux | Windows CMD | PowerShell |
|--------|-------|-------------|------------|
| Списък файлове | ls | dir | Get-ChildItem |
| Смяна директория | cd | cd | Set-Location |
| Преглед файл | cat | type | Get-Content |
| Копиране | cp | copy | Copy-Item |
| Преместване | mv | move | Move-Item |
| Изтриване | rm | del | Remove-Item |
| Създай директория | mkdir | mkdir | New-Item -Type Dir |
| Текуща директория | pwd | cd | Get-Location |
| Изчисти екран | clear | cls | Clear-Host |
| Процеси | ps, top | tasklist | Get-Process |
| Мрежа | ifconfig/ip | ipconfig | Get-NetIPAddress |

#### **Кога да използваме коя система:**

```
LINUX е по-добър за:
├── Сървъри (web, database, mail)
├── DevOps и Cloud
├── Програмиране и разработка
├── Мрежово оборудване
├── Embedded системи
├── Научни изчисления
└── Когато бюджетът е ограничен

WINDOWS е по-добър за:
├── Офис работа (MS Office)
├── Игри (DirectX, AntiCheat)
├── Специализиран софтуер (Adobe, CAD)
├── Enterprise среди (Active Directory)
├── Десктоп за не-техници
└── Хардуерна съвместимост (драйвери)
```
