#### **Част 2: Именуване на файлове (12 мин)**

**Windows правила:**
```
РАЗРЕШЕНИ:
- Букви (a-z, A-Z, Unicode)
- Цифри (0-9)
- Специални: _ - . (space)
- Максимална дължина: 255 символа (260 с пътя)

ЗАБРАНЕНИ СИМВОЛИ:
\ / : * ? " < > |

ЗАБРАНЕНИ ИМЕНА (запазени):
CON, PRN, AUX, NUL
COM1-COM9, LPT1-LPT9

CASE INSENSITIVE:
File.txt = FILE.TXT = file.txt
```

**Linux правила:**
```
РАЗРЕШЕНИ:
- Почти всичко освен / и NUL (null character)
- Максимална дължина: 255 символа

CASE SENSITIVE:
File.txt ≠ FILE.TXT ≠ file.txt

СКРИТИ ФАЙЛОВЕ:
Започват с точка: .bashrc, .config

ВНИМАНИЕ:
- Избягвайте интервали (използвайте _ или -)
- Избягвайте специални символи
```

#### **Част 3: Типове файлове и разширения (12 мин)**

```
РАЗШИРЕНИЕ = част след последната точка в името
file.txt      → разширение: txt
document.pdf  → разширение: pdf
archive.tar.gz → разширение: gz (или tar.gz)

COMMON EXTENSIONS:

Текстови:
.txt    Plain text
.md     Markdown
.html   HTML
.css    CSS stylesheet
.json   JSON data
.xml    XML data
.csv    Comma-separated values

Документи:
.doc/.docx   Microsoft Word
.xls/.xlsx   Microsoft Excel
.ppt/.pptx   Microsoft PowerPoint
.pdf         Portable Document Format
.odt         OpenDocument Text

Изображения:
.jpg/.jpeg   JPEG image (lossy)
.png         PNG image (lossless)
.gif         GIF (animated)
.svg         Scalable Vector Graphics
.bmp         Bitmap

Аудио/Видео:
.mp3         MPEG Audio Layer 3
.wav         Waveform Audio
.mp4         MPEG-4 video
.avi         Audio Video Interleave
.mkv         Matroska video

Архиви:
.zip         ZIP archive
.rar         RAR archive
.tar         Tape archive
.gz          Gzip compressed
.7z          7-Zip archive

Изпълними:
.exe         Windows executable
.msi         Windows installer
.bat/.cmd    Batch script
.sh          Shell script
.py          Python script
```

#### **Част 4: Файлови атрибути (8 мин)**

**Windows атрибути:**
```
R - Read-only (само за четене)
H - Hidden (скрит)
S - System (системен)
A - Archive (за архивиране)
I - Not content indexed
C - Compressed (компресиран)
E - Encrypted (криптиран)

Преглед: attrib filename
Задаване: attrib +R filename (добави)
         attrib -R filename (премахни)
```

**Linux атрибути:**
```
Показване: lsattr filename
Задаване: chattr +i filename (immutable)
         chattr -i filename

Атрибути:
i - immutable (не може да се променя)
a - append only (само добавяне)
c - compressed
e - extent format
```

### **4. ПРАКТИЧЕСКА РАБОТА (28 минути)**

**Задача 1: Изследване на разширения (10 мин)**
```
1. Създайте файлове с различни разширения:
   - test.txt, test.html, test.py

2. Опитайте да отворите всеки - каква програма се използва?

3. Преименувайте test.txt на test.jpg
   - Какво се случва? Може ли да се отвори?
   - Защо разширението е само индикатор?
```

**Задача 2: Атрибути (10 мин)**
```
Windows:
1. attrib test.txt
2. attrib +R test.txt
3. Опитайте да редактирате файла
4. attrib -R test.txt

Linux:
1. ls -la
2. file test.txt (показва реалния тип)
3. stat test.txt (пълна информация)
```

**Задача 3: Текстов vs Бинарен (8 мин)**
```
1. Създайте текстов файл с "Hello World"
2. Отворете го с Notepad/nano
3. Отворете .exe файл с Notepad
4. Каква е разликата?

Текстов: човешко-четим, ASCII/UTF-8
Бинарен: машинно-четим, специфичен формат
```

```
КЛЮЧОВИ ТОЧКИ:

1. Файлът = Метаданни + Данни
```
