```

   a) Striping  b) Mirroring  c) Parity  d) Concatenation

4. chmod 755 дава на собственика:
   a) r--  b) rw-  c) rwx  d) r-x

5. PaaS означава:
   a) Platform as a Service  b) Process as a Service
   c) Protocol as a Service  d) Product as a Service

6. Active Directory използва протокол:
   a) LDAP  b) FTP  c) SMTP  d) HTTP

7. NVMe използва интерфейс:
   a) SATA  b) PCIe  c) USB  d) Thunderbolt

8. Kubernetes е система за:
   a) Виртуализация  b) Оркестрация на контейнери
   c) Backup  d) Мониторинг

9. DDR5 RAM работи с напрежение:
   a) 1.5V  b) 1.35V  c) 1.2V  d) 1.1V

10. IaC означава:
    a) Internet as Code  b) Infrastructure as Code
    c) Interface as Code  d) Integration as Code
```
